Dokumentacja Projektu: Strona CV by Jakub Podleś 157748

Przegląd:
Ten projekt to strona CV dla Młodszego Specjalisty ds. E-commerce oraz Front-end Developera. Zawiera sekcje z podsumowaniem zawodowym, zestawami umiejętności, projektami portfolio oraz formularzem kontaktowym. Strona wykorzystuje Bootstrap do responsywnego designu oraz jQuery do elementów interaktywnych.

Struktura Plików:
- index.html: Główny plik HTML zawierający strukturę strony CV.
- styles/
  - main.css: Główny plik CSS, który importuje inne pliki CSS.
  - reset.css: Zawiera reguły resetu CSS.
  - base.css: Zawiera podstawowe style dla dokumentu.
  - nav.css: Zawiera style związane z paskiem nawigacyjnym.
  - sections.css: Zawiera style dla różnych sekcji strony.
  - popups.css: Zawiera style dla popupów.
- scripts/
  - main.js: Zawiera wszystkie skrypty jQuery dla elementów interaktywnych.

Struktura HTML:
- Plik HTML zawiera następujące główne sekcje:
  - Pasek Nawigacyjny: Zawiera przyciski do nawigacji.
  - Środkowy Div: Zawiera główne podsumowanie i obrazek.
  - Opis: Zawiera animowany tekst wpisywany.
  - Tabela Div: Zawiera szczegóły dotyczące umiejętności i doświadczenia.
  - Projekty: Zawiera informacje o projektach portfolio.
  - Stopka: Zawiera informacje stopki.

Podział CSS:
1. reset.css:
   - Resetuje domyślne style przeglądarki dla spójności.

2. base.css:
   - Ustawia globalną rodzinę czcionek i ogólne style dla body i wspólnych elementów.

3. nav.css:
   - Style dla paska nawigacyjnego, w tym przyciski i efekty najechania.

4. sections.css:
   - Style dla głównych sekcji, w tym center-div, description, tablediv, projects i footer.

5. popups.css:
   - Style dla popupów, w tym układ, elementy formularza i responsywność.

JavaScript:
- Projekt używa jQuery do obsługi interaktywnych elementów, takich jak popupy i wysyłanie formularza.
- main.js zawiera następującą funkcjonalność:
  - Wyświetla losowe ciekawostki dotyczące kodowania w popupie po kliknięciu przycisku "Random Fun-Fact".
  - Wyświetla formularz e-mail w popupie po kliknięciu przycisku "Send me an e-mail".
  - Obsługuje zamykanie popupów.

Biblioteki i Frameworki:
1. Bootstrap:
   - Używany do responsywnego designu i stylizacji elementów.
   - Wersja: 5.3.3
   - Linki CDN zawarte w pliku HTML.

2. jQuery:
